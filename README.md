# Pavel's Digital Solutions Website

One-page business website for Pavel's Digital Solutions.

## Included

- `index.html`
- `style.css`
- `script.js`
- `assets/reference-flyer.jpg`

## Open locally

Open `index.html` in a browser, or serve the folder with a local static server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.
